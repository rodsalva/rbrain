## O que está aqui

Decisões de arquitetura, stack técnico, integrações, AI/ML, infraestrutura e escolhas
de build vs. buy. É o repositório de como o produto é construído e por quê.

## Sub-tópicos

- **architecture** — Decisões de arquitetura, data models, API design
- **integrations** — Integrações com sistemas externos (SPED, eSocial, ERP, etc.)
- **ai-ml** — Implementação de AI/ML, modelos, embeddings, pipelines
- **infrastructure** — Infra, deploy, cloud, banco de dados
- **build-vs-buy** — Análises de construir vs. comprar componentes
